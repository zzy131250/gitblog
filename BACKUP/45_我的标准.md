# 完美的工作 todo
完美的工作可能不存在，但是对完美工作的标准可以存在😏
- 工作难度：有些挑战，但与能力匹配，不是过于简单或者过于困难
- 工作时间：朝九晚五的工作才是好工作
- 工作氛围：简单，友好，大家只是为了工作聚在一起
- 工作回报：中偏上即可，能够覆盖日常开销，并有些闲钱用来享受
